# EndoCare - Prenotazioni Endoscopia

## Descrizione

Applicazione web full-stack API-based sviluppata per la gestione delle prenotazioni di esami endoscopici.

Il progetto è stato realizzato utilizzando:

- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQLite
- Architettura: REST API

## Funzionalità

- Creazione prenotazioni
- Visualizzazione prenotazioni
- Eliminazione prenotazioni
- Controllo slot occupati
- Blocco prenotazioni domenicali
- Persistenza dati tramite SQLite

## Struttura del progetto

- app.py → Backend Flask
- index.html → Interfaccia utente
- script.js → Logica frontend
- style.css → Stile grafico
- database.db → Database SQLite
